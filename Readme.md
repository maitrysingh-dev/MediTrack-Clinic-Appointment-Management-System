MediTrack - Healthcare Management Console Application
Overview

MediTrack is a Java-based console application for managing patients, doctors, appointments, and billing.

Features
Add/View Patients
Add/View Doctors
Book/Cancel Appointments
Generate Bills
CSV Save/Load Persistence
Input Validation
Custom Exception Handling
Generic DataStore
Immutable BillSummary
Enums for Appointment Status and Specialization
Tech Stack
Java 17+
IntelliJ IDEA
CSV File Handling
OOP + Collections + Generics

entity
service
util
exception

constants

test
How to Run
Clone repository
Open in IntelliJ
Run Main.java
Use menu-driven console

Use menu options to save/load CSV data.
